LangChain is a framework designed to facilitate the creation of applications using large language models (LLMs). 
It provides tools for document loading, text splitting, embeddings, and vector stores.

RAG (Retrieval-Augmented Generation) is a technique that gives an LLM access to external data. 
This allows the model to answer questions with high accuracy and reduced hallucinations.
